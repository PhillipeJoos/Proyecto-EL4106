## Installation

You must install the requirements on a python environment.

If you are on linux:

```bash
cd ~
git clone https://github.com/PhillipeJoos/Proyecto-EL4106.git
cd ~/Proyecto-EL4106
mkdir -p ~/Proyecto-EL4106/img
python3 -m venv venv
source venv/bin/activate
pip install -r requirementsLinux.txt
```

## How to run

```bash
python3 main.py
```
